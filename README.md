### Used Versions
- Laravel v6.0.3
- Laravel Passport v7.4.1
- Laravel Ui v1.0.1
- Laravel Permission v3.0

### To Use
- create .env file
- create database
- php artisan key:generate
- npm install
- npm run dev
- composer install
- php artisan migrate:refresh
- php artisan passport:install
- php artisan db:seed
